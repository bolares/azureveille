Azure

Qu'est ce que c'est Microsoft Azure ?

Lancé en 2010 Azure est la plateforme cloud de Microsoft. Elle regroupe divers services de cloud Computing. On y trouve à la fois des solutions open source que des technologies proprietaire de Microsoft ou d'autres entreprises. A l'origine, la plateforme s'appelait Windows Azure

C'est quoi le cloud computing ?

Retenez tout simplement que le cloud computing est la fourniture de services informatiques tels que la gestion des réseaux, des logiciels, des outils d'analyse, de stockage, de serveurs, d'intelligence artificielle via internet (cloud)

Pourquoi Microsoft Azure ?

La solution Azure a été conçu pour gérer toutes les charges de travail. Azure propose des contrats SLA(Service-level-agreement), ainsi qu'une assistance technique et une analyse du fonctionnement des services en continu.
